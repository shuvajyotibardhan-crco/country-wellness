# **Public Sources for International Comparison Metrics**

This guide lists the primary, authoritative, and publicly accessible sources for the most widely accepted international country metrics.

## **1\. Economic Metrics**

| Metric | Primary Custodian / Source | Where to Access the Data (Public Link) |
| :---- | :---- | :---- |
| **Gross Domestic Product (GDP)** | **The World Bank** (World Development Indicators) | [World Bank Open Data](https://data.worldbank.org/) |
| **GDP per Capita** | **The World Bank** | [World Bank GDP per Capita (Current USD)](https://data.worldbank.org/indicator/NY.GDP.PCAP.CD) |
| **Purchasing Power Parity (PPP)** | **International Comparison Program (ICP)** / World Bank | [World Bank GDP, PPP (Current International $)](https://data.worldbank.org/indicator/NY.GDP.MKTP.PP.CD) |
| **Gini Coefficient (Income Inequality)** | **The World Bank** (Development Research Group) | [World Bank Gini Index](https://data.worldbank.org/indicator/SI.POV.GINI) |

### **Quick Example:**

If you want to compare how rich the average citizen of two countries is while adjusting for local prices (like how much a cup of coffee costs in each country), download the **GDP per capita, PPP** dataset directly from the World Bank.

## **2\. Social & Human Development Metrics**

| Metric | Primary Custodian / Source | Where to Access the Data (Public Link) |
| :---- | :---- | :---- |
| **Human Development Index (HDI)** | **United Nations Development Programme (UNDP)** | [UNDP Human Development Reports Data Center](https://hdr.undp.org/data-center) |
| **Life Expectancy at Birth** | **United Nations Population Division** / World Bank | [World Bank Life Expectancy Data](https://data.worldbank.org/indicator/SP.DYN.LE00.IN) |
| **Literacy Rate** | **UNESCO Institute for Statistics (UIS)** | [UNESCO UIS Data Portal](http://uis.unesco.org/) |

### **Quick Example:**

To see if a country's overall living standards are improving beyond just economic growth, go to the UNDP Data Center to look up its **HDI** value (which balances life expectancy, education, and income).

## **3\. Governance & Stability Metrics**

| Metric | Primary Custodian / Source | Where to Access the Data (Public Link) |
| :---- | :---- | :---- |
| **Corruption Perceptions Index (CPI)** | **Transparency International** | [Transparency International CPI Portal](https://www.transparency.org/en/cpi) |
| **Global Peace Index (GPI)** | **Institute for Economics and Peace (IEP)** | [Vision of Humanity Data Portal](https://www.visionofhumanity.org/) |

### **Quick Example:**

If a business is trying to decide if it is safe to open a branch in a specific country, they will check the **Corruption Perceptions Index** on Transparency International to see how clean or corrupt the local public sector is.

## **4\. Environmental & Wellbeing Metrics**

| Metric | Primary Custodian / Source | Where to Access the Data (Public Link) |
| :---- | :---- | :---- |
| **Environmental Performance Index (EPI)** | **Yale University** / **Columbia University** | [Yale EPI Website](https://epi.yale.edu/) |
| **World Happiness Report** | **Sustainable Development Solutions Network (SDSN)** | [World Happiness Report Data](https://worldhappiness.report/) |

### **Quick Example:**

To compare how successfully different governments are tackling air pollution and climate change, download the detailed scorecard sheets directly from the Yale **EPI** database.
