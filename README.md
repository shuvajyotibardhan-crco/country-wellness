# Country Wellness Check

A single-page web app that compares up to four countries across 11 wellness KPIs spanning economic, social, governance, and environmental dimensions.

## Setup

No build step required. Open `index.html` in a browser, or serve via any static host.

## Data Sources

All data is pulled live from public APIs (World Bank) or from embedded static datasets curated from official sources (UNDP, Transparency International, IEP, Yale EPI, World Happiness Report, UNESCO).

## Features

- Select 1–4 countries via searchable dropdown (3-character typeahead)
- Side-by-side KPI comparison table
- Optional year range → trend line chart per KPI per country
- Source citations at page bottom
